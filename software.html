Software

Open-source software that I developed (or am currently developing) and help manage as part of two GitHub "organizations".

MarEcoTel

  • ZcTransCMR
    Bayesian continuous-time open-population mark-recapture modeling framework. Tools for estimating survival, residency, and movement using continuous-time CMR data.
  • WCtagproc
    A suite of tools for processing Wildlife Computers satellite tag datasets, including data cleaning, filtering, and standardized extraction workflows.

animaltags

  • Software for High-Resolution Animal-Borne Tag Data
    Contributor to cross-platform tools supporting data from high-resolution movement recorders used in behavioral and ecological research.
  • GitHub Organization Developer
    R, MATLAB, and Octave code development for onboard sensor calibration, data processing, and analysis.
